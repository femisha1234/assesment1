iframe Tag in HTML

The iframe tag in HTML is used to embed another HTML document within the current document. This allows you to display content from another webpage, such as a video, map, or any external website, directly within your webpage.

Common Attributes of the iframe Tag

  • src: Specifies the URL of the document to be embedded.
  • width: Specifies the width of the iframe (in pixels or percentage).
  • height: Specifies the height of the iframe (in pixels or percentage).
  • name: Specifies the name of the iframe (used for targetting links).
