Heading Tags in HTML

  • h1 Tag-This tag defines the most important heading on the page.
  • h2 Tag-This tag defines a subheading under h1.
  • h3 Tag-This tag defines a subheading under h2.
  • h4 Tag-This tag defines a subheading under h3.
  • h5 Tag-This tag defines a subheading under h4.
  • h6 Tag-This tag defines a subheading under h5.

Example:-

Document Title

Section 1

Some introductory text for section 1.

Subsection 1.1

Details for subsection 1.1.

Subsection 1.2

Details for subsection 1.2.

Section 2

Some introductory text for section 2.

Subsection 2.1

Details for subsection 2.1.

Subsection 2.2

Details for subsection 2.2.
