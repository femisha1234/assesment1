Styling Tag in HTML

In HTML, the style tag is used to define CSS (Cascading Style Sheets) styles within an HTML document. This tag is placed within the head section of the document and contains CSS rules that apply styles to various HTML elements.

Inline vs. Internal vs. External CSS

  • Inline CSS: Styles are applied directly within HTML elements using the style attribute. Example:

    This is an inline styled paragraph.

  • Internal CSS: Styles are defined within the style tag inside the head section.


  • External CSS: Styles are defined in an external CSS file, which is linked to the HTML document using the link tag.
