Github

GitHub is a web-based platform that uses Git for version control and offers a suite of tools for collaboration and project management. It is widely used for hosting and sharing code repositories, managing projects, and collaborating with other developers.

  • Repositories: A repository (or "repo") is where your project resides. It contains all your project files and each file's revision history.
  • Branches: Multiple versions of a project can be developed simultaneously using branches.
  • Pull Requests (PRs): A pull request is a way to propose changes to the repository. It allows for code review and discussion before merging changes.
  • Issues: GitHub Issues are used to track tasks, enhancements, bugs, and other project-related work.
  • Actions: GitHub Actions automate workflows for continuous integration and continuous deployment (CI/CD).

    • Creating a GitHub Account

  • Go to GitHub's website. Click "Sign up" and follow the instructions to create an account.

    • Creating a New Repository

  • After logging in, click the + icon in the top right corner and select "New repository". Fill in the repository name and description. Choose to make the repository public or private. Optionally, initialize the repository with a README, .gitignore, and license. Click "Create repository".
